# Arduino

该项目演示如何使用Arduino来存储任意组合逻辑

## 硬件 :scissors:

- Arduino Nano
- EEPROM AT28C16
- 74HC595

## 软件 :computer:

- Arduino IDE (一般会自动把编译器和对应的驱动下载好)
- [CH340G驱动](https://www.wch.cn/products/CH340.html) 或 [FT232R驱动](https://ftdichip.com/drivers/vcp-drivers/)
